Random CSS Background Rotator

 

This page was created to demonstrate how the PHP rotator script by Automatic Labs can be used to rotate random CSS background images. As you refresh this page, you should be seeing one of several different possible header images. If the page is refreshed too quickly, sometimes the background will repeat itself, but for the most part your browser should display a different image for every page load.

XHTML CSS 508 | Example and tutorial by Nathan Smith. | PHP rotator script by Dan Benjamin.
